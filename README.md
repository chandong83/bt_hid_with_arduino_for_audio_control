##Joystick connection MAP

>GND    -    GND

>+5V     -    5V

>VRx     -    A0

>VRy     -    A1

>SW      -   N.C(사용 안 함)



##FB155BC connection MAP
>1.GND      -  GND

>2.VCC       -  3.3V

>3.STATUS  -  7

>4.MULTI    -  5

>5.NC        -  NC

>6.HID ST  -  NC

>7.TXD      -  2

>8.RXD      -  3


##Other connection MAP
>LED +   -   4

>LED -    -   GND
